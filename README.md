plot_gen
====

物語の格子をランダムで出力します

## Description

大塚英志著の「物語の体操」に示されている、
カードを用いて行う物語の解体と再構成のレッスンを簡易に行えるよう、
自動で結果を出力するプログラムを作成しました。

## Demo

ruby ./plot_gen.rb

結果例.

"主人公の現在: 善良 : 正位置"
"主人公の近い未来: 知恵 : 逆位置"
"主人公の過去: 誓約 : 正位置"
"援助者: 慈愛 : 正位置"
"敵対者: 秩序 : 逆位置"
"結末: 調和 : 逆位置"
